# Syntax

## Links
### Internal / Wikilinks
#### Internal Links
Intrnal links to other wikipages are created like a normal link but prefixed with a colon.

``` md
This is a internal link [Link Text](:pagelink).
```

#### Wikilinks 

Wikilinks are also supported:

``` md
This is also a internal link [[pagelink]].
```
